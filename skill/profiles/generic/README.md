# Profile: generic

Minimal placeholder profile. Use this when:

- You're evaluating the skill and don't want to commit to an LMS yet
- Your LMS is not yet supported by any of the other profiles
- You publish materials manually (email, classroom share drive, internal portal)

This profile has **no opinions** about institution, audience, or LMS — every
`course init` question is asked fresh. The `lms.md` workflow is "build a zip
and upload to wherever".

If you find yourself answering the same `course init` questions across
multiple courses, copy this profile to `<your-slug>/` and fill in
`course_defaults.yaml`. See `skill/profiles/README.md`.
